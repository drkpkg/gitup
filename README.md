# GitUp, a simple GitHub uploader

## Usage (simple as 1, 2, 3)

- Install gitup:

```sh
$ sudo ./installer
```

- Go to your repository folder:

```sh
$ cd /home/user/your/repository
```

- Execute GitUp

```sh
$ gitup
```

- Insert your commit message

- Hit ENTER

- Done!
